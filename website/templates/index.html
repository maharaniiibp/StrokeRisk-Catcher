Mengapa CNN 1D?

Model ini menggunakan arsitektur Convolutional Neural Network 1-Dimensi untuk menangkap pola dan korelasi antar parameter klinis pasien.

✅

Ekstraksi Fitur Otomatis

Menangkap korelasi antar variabel klinis tanpa perlu rekayasa fitur manual.

🛡️

Robust terhadap Noise

Lebih stabil terhadap variasi data pasien dan mengurangi dampak outlier.

📈

Recall Tinggi

Dioptimalkan untuk meminimalkan kasus stroke positif yang terlewat.

Arsitektur: 1D Convolution + Batch Normalization + Dropout, diikuti Global Max Pooling dan Fully Connected Layer dengan aktivasi sigmoid untuk klasifikasi biner.

Catatan: sesuaikan detail arsitektur di atas dengan struktur model kamu yang sebenarnya (jumlah layer, filter, dsb dari train_cnn.py).

Spesifikasi Model

Arsitektur: Deep Learning 1D CNN dengan Batch Normalization dan Dropout layers untuk regularisasi.

Dataset: Cerebral Stroke Prediction (43.400 data awal), mencakup parameter klinis: umur, hipertensi, penyakit jantung, status merokok, dan kadar glukosa.

Optimasi: Binary Cross-Entropy Loss dengan Adam Optimizer. Early Stopping diterapkan untuk mencegah overfitting.
